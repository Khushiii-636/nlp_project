**Project Overview**
E-commerce platforms generate massive amounts of customer reviews daily. These reviews contain valuable insights about product quality, delivery, pricing, and customer satisfaction.

This project builds an Explainable Aspect-Based Sentiment Analysis System using a transformer model (BERT) to:

Extract product aspects
Predict sentiment for each aspect
Provide interpretable outputs

Traditional sentiment analysis:
Only gives overall sentiment
Cannot detect multiple opinions in one review
Lacks explainability

**Example:**
"Delivery was late but product quality is excellent"
Traditional output → Neutral
Our system →
Delivery → Negative
Quality → Positive

**Objectives**
Perform Aspect-Based Sentiment Analysis (ABSA)
Use BERT transformer model for better accuracy
Provide explainable outputs

**Key Features**
Aspect Extraction (Price, Quality, Delivery, etc.)
BERT-based Sentiment Classification
Fake Review Detection
Review Summarization
Trend Analysis
Explainable Outputs
Help:
Customers make better decisions
Businesses improve specific features
